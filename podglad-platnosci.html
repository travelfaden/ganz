🔧 Obecna Konfiguracja

Backend (backend-example.js)

app.post('/api/create-checkout-session', async (req, res) => { const session = await stripe.checkout.sessions.create({ payment_method_types: ['card'], line_items: [{ price_data: { currency: 'eur', product_data: { name: productName, description: 'Voucher podróżniczy...' }, unit_amount: amount * 100 }, quantity: 1 }], mode: 'payment', success_url: '.../success.html', cancel_url: '.../cancel.html', custom_text: { submit: { message: 'Dziękujemy za wybór Travel Faden!' } } }); });

Frontend (test-stripe.html)

// Tworzenie sesji przez backend const response = await fetch('http://localhost:3000/api/create-checkout-session', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ amount, currency: 'eur', productName }) }); // Przekierowanie do Stripe Checkout const session = await response.json(); stripe.redirectToCheckout({ sessionId: session.id });

🎨 Możliwości Customizacji

Co można zmienić TERAZ (bez dodatkowego kodu):

  • ✅ Kolory - Stripe Dashboard → Settings → Branding
  • ✅ Logo - Stripe Dashboard → Settings → Branding
  • ✅ Teksty - przez custom_text w backendzie (już dodane)

Co można zmienić z dodatkowym kodem:

  • 🔧 Pełna kontrola designu - użyj Stripe Elements (przykład: stripe-elements-example.html)
  • 🔧 Własny formularz - zbuduj formularz na swojej stronie
  • 🔧 Dodatkowe pola - adres, telefon, itp.
